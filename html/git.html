Git

  • git init - создание нового репозитория.
  • git clone - клонирование удаленного репозитория.
  • git status - ппросмотр текущего состояния файлов и репозитория.
  • git add - добавление в отслеживаемые файлы.
  • git commit -m - сохранение изменений.
  • git push - внесение изменений на удаленный репозиторий.
  • git log - отображение истории коммитов.
  • git pull - получение изменений из удаленного репозитория.
  • git branch - создание новой ветки.
  • git checkout - переключение на указанную ветку.
